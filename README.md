== Fields ==

 - Title (Lable as Name)
 - Description
 - Image (Photo)
 - Contact details
   - Email
   - Phone
   - Skype
   - IM
   - Street Address
 - Country
 - Region
 - Specialization - General (Vocabulary)
   Custom fields can be added via behaviours as needed.
 - Comments enabled

== Types allowed in container ==

 - File (for CV)
 - Image

== View ==


In addition to fields, and list of contained objects, list of existing Contracts, start/end/status of ploneun.tor:

 - Back-links to existing ploneUN.TOR contracts (dates, state) (Optional)

ploneun.tor is optional, consultant can be installed standalone.
